# 365new 작업지시서 웹앱 시스템

## 개요
석면 해체·철거 전문 업체의 현장 작업지시서 입력 및 블로그 원고 자동 생성 시스템

## 주요 기능
- 작업지시서 웹앱 입력 (영업사원용)
- 프로젝트번호 자동 발급
- 작업신고 담당자 이메일 자동 발송
- AI 블로그 원고 자동 생성 (Claude API)
- 원가계산 초안 자동 산출
